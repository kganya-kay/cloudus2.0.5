Platform Capabilities (Routes, Processes, Actions)
Public & Marketing
- / : Landing shell with navigation to Shop, Projects, Rentals, Feed, Suppliers, Drivers, Creators, Calendar, Careers, Team.
- /shop : Marketplace of packaged services; browse cards, open /shop/[itemId]; create orders; pay via Paystack checkout/webhook.
- /projects : Project marketplace; /projects/create to publish briefs; /projects/[projectId] for detail, bids, tasks, payments.
- /feed : Creator/marketplace feed with media, reactions, admin review queue.
- /rentals (alias /rooms) : Rentals marketplace listing; /rentals/[roomId] detail with gallery, nightly + monthly rates, booking form, Paystack checkout; /rentals/create for host submission (admin approval).
- /team : Meet squads; /careers : job listings and apply flow; /privacy, /terms, /specials : informational/legal pages.

Auth & Profile
- /auth/login, /auth/register (NextAuth); /profile/[email] for profile editing.

Commerce & Orders
- Shop orders: Detail at /shop/[itemId], create orders with optional delivery; payments via /api/payments/paystack/checkout; webhook updates Payment status; fulfillment tracked with caretakers/suppliers/drivers.
- Rentals bookings: /rooms/[roomId] booking creates Booking + BookingPayment; Paystack checkout at /api/bookings/paystack/checkout; webhook confirms and marks booking CONFIRMED/PAID or CANCELED.
- Laundry: /laundry flow with payment success at /laundry/payment/success.

Projects & Collaboration
- /projects/create to launch; /projects/[projectId] for bids, tasks, comments, likes/upvotes, payments; payouts and collaboration invites managed via project router.

Creators & Feed
- /creators/dashboard : Creator hub, featured creators, follower/following, feed posts with media.

Suppliers & Drivers
- /suppliers/apply onboarding; /suppliers/dashboard operations (orders, payouts, GPS).
- /drivers/apply onboarding; /drivers/dashboard for delivery ops/status/location.

Calendar & Scheduling
- /calendar : Events/tasks; includes Salesforce event sync.

Careers
- /careers : Job listings; applications capture resume, cover letter, metadata.

Admin (Role: ADMIN/CARETaker unless noted)
- /admin : Ops overview (announcements, review queue, quick links).
- Quick links: /admin/suppliers, /admin/drivers, /admin/orders, /admin/shopitems, /admin/reports, /admin/users, /admin/applications, /admin/rooms.
- /admin/rooms : Review rentals; filter by Pending/Approved/Rejected/All; approve/reject sets adminStatus, activates listings, stamps publishedAt.
- Other admin pages manage suppliers, drivers, orders, shop items, users, applications, reports; review queue covers feed/projects/shop items.

Rental Workflows
- Host: /rooms/create → PENDING → /admin/rooms approval → APPROVED sets isActive, publishes to /rooms + /rentals.
- Guest: /rooms/[roomId] → enter dates/guests → booking created if no overlap → Paystack checkout → webhook confirms booking.

AI Navigator
- Assistant input uses npm AI SDK (`generateText` with model "openai/gpt-5") with sitemap-grounded prompt to return paths/actions; guides users across routes.

Data & Reporting
- Captures orders, projects, bookings, deliveries, payouts; /admin/reports for ops; Power BI-ready outputs; CRM touchpoints emphasized (“A growing business needs to know every customer touchpoint”).

- Auth: NextAuth with roles (ADMIN, CARETAKER, SUPPLIER, CUSTOMER, DRIVER).
- AI: npm AI SDK for navigator.
- Cloud-ready: Azure-compatible deployment; supports databases, APIs, and BI integrations.
